## Acerca del proyecto

Este proyecto especial es una expresión sincera de amor y creatividad creada para una novia querida. Desarrollado con React, el proyecto ofrece una experiencia dinámica y personalizada que plantea una pregunta única para el Día de San Valentín mediante una combinación de gifs y mensajes. El sitio web, adornado con la fuente romántica "Lobster", invita a la persona a aceptar o rechazar la propuesta. Al aceptar, aparece un mensaje encantador con un gif tierno y notas de amor personalizadas, creando una experiencia cálida y memorable.

### Alojamiento

El frontend de la aplicación está alojado en Netlify.
**Puedes ver el sitio en vivo en este enlace:
<a href="https://jocelynsvalentinesurprise.netlify.app/" target="_blank">Un mensaje especial</a>**

### Tecnologías principales

- React: Biblioteca de JavaScript para crear interfaces de usuario.
- HTML: Lenguaje de marcado estándar para crear páginas web.
- CSS: Hojas de estilo en cascada, usadas para el diseño visual de la aplicación.

## Comenzando

Para configurar el proyecto localmente, sigue estos sencillos pasos.

### Requisitos previos

_Software necesario para ejecutar el proyecto._

- [Visual Studio Code](https://code.visualstudio.com/)
- [Node/Npm](https://nodejs.org/en)
- [Git](https://git-scm.com/)

### Instalación

_Cómo instalar y preparar el proyecto._

Desde la línea de comandos:

```bash
# Clonar este repositorio
git clone https://github.com/AlexisRodriguezCS/ValentineProposal.git

# Entrar en la carpeta del proyecto
cd ValentineProposal

# Instalar dependencias
npm install

# Iniciar la aplicación
npm start
```
